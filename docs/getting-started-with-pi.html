3.1 How to set up your Pi

3.1.0.1 Install the Raspbian on your SD card

Step 1: Install the Pi OS on your SD card

  1. Download the Raspberry Pi Imager

  2. Plug in your SD card on your laptop.

  3. Open the imager and format your SD card.

  4. Write the Raspberry Pi OS on your SD card. Please enable ssh and set username (pi) and password (raspberry) in the setting before the writing.
